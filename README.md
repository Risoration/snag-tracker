# Snag Tracker Starter

Fresh Next.js 14 project bootstrapped for Homes by Honey. It ships with the App Router, TypeScript, ESLint, Tailwind CSS, and an alias for `@/*` pointing at `src`.

## Getting Started

```bash
pnpm install
pnpm dev
```

Then open [http://localhost:3000](http://localhost:3000) to view the local build.

## Project Layout

- `app/` – App Router entry points and global styles.
- `src/` – Place shared UI, hooks, lib utilities, and types.
- `tailwind.config.ts` / `postcss.config.mjs` – Tailwind + PostCSS configuration.
- `next.config.mjs` – Next.js project config hook.

Happy building and snag reporting!
